使用 Docker 容器化 Web应用

发表于2023-11-22|更新于2024-06-25|编程

https://www.docker.com/products/docker-desktop/

为什么要用 Docker

  1. Docker 可以提供一个稳定的“Linux”的开发环境
  2. Web应用，大都部署在Linux服务器上，使用Docker可以模拟这样的环境
  3. 使用Docker可以进行容器编排，有助于开发微服务
  4. 使用Docker可以快速高效的配置 python、anaconda、Redis、sqlserver、mysql等诸多复杂的开发环境
  5. 可以打拉取镜像，打包镜像，方便快捷

其他

在 相同 Docker-Compose 容器编排下内部之间调用可以直接 http://service_name:内部端口号，不用从外部调用。
