Git管理Unity项目

发表于2023-10-09|更新于2023-10-15|Unity

Unity设置

Unit——Project Settings——Version Control———Mode：Visible Meta Files

Unit——Project Settings——Editor——Mode:Force Text

配置.gitignore文件

Unity 项目除”Assets”、”ProjectSettings”文件夹外其余文件均为Unity生成，将其排除
最好在初始化之前或者在第一次提交之前编写好”.gitignore”文件

https://github.com/github/gitignore/blob/main/Unity.gitignore

git基础命令:

本地：

git init
git staus
git add ./
git commit -m "message"
git log
git reset --hard HEAD^

远程：

ssh-keygen -t rsa -C "邮箱"
git remote add origin 远程仓库地址
git branch -M main
git push -u origin main

克隆:

git clone 远程仓库地址

GIthub Desktop

如果使用 GitHub 项目名不要使用空格，用破折号或者下划线代替 * dotnet 项目用点*

可以使用 Github Desktop 个人喜欢使用 vscode 或者 visual studio 的git

使用 Github Desktop 的情况下：

仓库名和项目名相同，本地路径使用上一层路径

eg:`..\Unity Projects\Github-Project`  
Github-Project：项目名也是仓库名
Unity Projects：本地路径

Git ignore 选择Unity

Discard all changes..:可以撤销所有未提交的改动

publish repository:推送到Github远程库
